## 0. Краткое резюме (Executive Summary)

Проектирование и разработка нового сайта стоматологической клиники **ООО «Моя Стоматология»** (Сочи, Донская 52). Проект ведётся на бартерной основе через брата исполнителя — зуботехника, эксклюзивно работающего с клиникой.

**Главная цель** — максимизировать поток первичных пациентов из органического поиска через комплексную SEO-стратегию: оптимизация под Яндекс/Google, идеальные Core Web Vitals, schema.org-разметка, контент-маркетинг через блог.

**Стек** — Next.js 15 со статическим экспортом для размещения на собственном сервере владельца.

**Целевые SEO-кластеры**: протезирование, имплантация, эндодонтия под микроскопом (уникальное УТП), общая терапия и хирургия.

**Существующий сайт** mydentsochi.ru сломан и подлежит замене.

**Текущий рейтинг клиники**: ★ 4.9 на 2ГИС (104 отзыва), ★ 4.7 на Яндекс.Картах (40 отзывов).

## 2. Цели и метрики успеха

### 2.1. Бизнес-цели
- **Главная**: максимизация потока первичных пациентов из органического поиска
- **Подцель**: увеличение записей на ортопедию и имплантацию (приоритет — больше работ для брата-техника)

### 2.2. SEO-метрики
- **Через 6 месяцев**: топ-10 Яндекса/Google по 15–25 низко- и среднечастотным запросам
- **Через 12 месяцев**: топ-10 по 50+ запросам, включая высокочастотники типа «имплантация зубов Сочи»
- **Технические**: Lighthouse ≥ 95 на всех ключевых страницах
- **Поведенческие**: bounce rate < 50%, среднее время на странице услуги > 1.5 мин
- **Конверсия**: ≥ 3% посетителей оставляют заявку на запись

### 2.3. Не-цели (out of scope для текущей версии)
- Приём по острой боли вне очереди — клиника не принимает
- Детская стоматология — на сайте не продвигается (нет подтверждённого детского врача)
- Личный кабинет пациента
- Мобильное приложение
- Английская версия сайта (отложено до Phase 3)
- Онлайн-оплата на сайте

## 4. Технологический стек

### 4.1. Frontend
- **Next.js 15** (App Router)
- **Режим сборки**: `output: 'export'` (статический экспорт → загрузка папки на nginx)
- **TypeScript** (strict mode)
- **TailwindCSS** + shadcn/ui
- **MDX** для блога (через `@next/mdx`)

### 4.2. Формы и валидация
- **react-hook-form** + **zod**
- Отправка формы записи → **Telegram-бот владельца** (через webhook на собственном микро-эндпоинте либо напрямую в Bot API)
- Без баз данных на этапе MVP

### 4.3. SEO-инструменты
- **next-sitemap** — автогенерация sitemap.xml + robots.txt
- Генерация мета-тегов через нативный `generateMetadata` (App Router)
- Кастомный модуль JSON-LD для schema.org (типизированные builders по типам страниц)

### 4.4. Изображения
- **next/image** с кастомным `loader` (т.к. статический экспорт)
- WebP / AVIF
- Lazy loading ниже первого экрана
- Fixed dimensions у всех картинок (CLS = 0)

### 4.5. Аналитика
- **Яндекс.Метрика** (приоритет — основной поиск в РФ)
- Google Analytics 4 — опционально

### 4.6. Деплой
- `npm run build` → папка `out/` (статика)
- Загрузка папки на собственный nginx-сервер заказчика
- Конфигурация nginx: gzip + brotli, кэширование статики, HTTP/2, fallback `try_files $uri $uri/ /404.html`
